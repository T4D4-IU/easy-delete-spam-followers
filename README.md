# Twitter Follower Manager

Twitterのフォロワー欄から直接フォロワーを解除またはブロックできるChrome拡張機能です。

## 機能

- **フォロワー解除**: フォロワーをブロック→即座にブロック解除することで、フォロワーから外します
- **ブロック**: ユーザーをブロックします

## インストール方法

1. Chrome拡張機能の管理画面を開く
   - `chrome://extensions/` にアクセス

2. 「デベロッパーモード」を有効にする（右上のトグル）

3. 「パッケージ化されていない拡張機能を読み込む」をクリック

4. このフォルダ（`twitter-follower-manager`）を選択

## 使い方

1. Twitterにログイン

2. 自分のプロフィール→「フォロワー」タブを開く
   - URL: `https://twitter.com/[あなたのユーザー名]/followers`

3. 各フォロワーに「解除」と「ブロック」ボタンが表示されます

4. ボタンをクリックして操作を実行

## 注意事項

- この拡張機能はTwitterの非公式APIを使用しています
- Twitter/Xの仕様変更により動作しなくなる可能性があります
- フォロワー解除は「ブロック→即座にブロック解除」という方法で実現しています
- 大量に連続使用するとアカウント制限を受ける可能性があります

## アイコンについて

この拡張機能を使用するには、以下のサイズのアイコン画像を用意してください：
- icon16.png (16x16px)
- icon48.png (48x48px)
- icon128.png (128x128px)

アイコンがない場合でも機能は動作しますが、警告が表示されます。
