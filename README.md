# Twitter Follower Manager

X (旧Twitter) のフォロワー欄から直接フォロワーを解除またはブロックできるChrome拡張機能です。

## 機能

- **フォロワー解除**: フォロワーをブロック→即座にブロック解除することで、フォロワーから外します
- **ブロック**: ユーザーをブロックします

## インストール方法

1. このリポジトリをクローンまたはダウンロード
   ```bash
   git clone https://github.com/T4D4-IU/twitter-follower-manager.git
   ```

2. Chrome拡張機能の管理画面を開く
   - `chrome://extensions/` にアクセス

3. 「デベロッパーモード」を有効にする（右上のトグル）

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダ（`twitter-follower-manager`）を選択

## 使い方

1. X (旧Twitter) にログイン

2. 自分のプロフィール→「フォロワー」タブを開く
   - URL: `https://x.com/[あなたのユーザー名]/followers`
   - または: `https://twitter.com/[あなたのユーザー名]/followers`

3. 各フォロワーの右側に「解除」と「ブロック」ボタンが表示されます

4. ボタンをクリックして操作を実行
   - **解除**: 確認ダイアログが表示され、OKをクリックするとフォロワーから解除されます
   - **ブロック**: 確認ダイアログが表示され、OKをクリックするとユーザーをブロックします

## 注意事項

- この拡張機能はX/Twitterの内部APIを使用しています
- X/Twitterの仕様変更により動作しなくなる可能性があります
- フォロワー解除は「ブロック→即座にブロック解除」という方法で実現しています
- 大量に連続使用するとアカウント制限を受ける可能性がありますので、適度な間隔をあけて使用してください
